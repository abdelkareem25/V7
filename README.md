E-Commerce REST API | ASP.NET Core, EF Core, SQL Server, Redis, Stripe, JWT

Built a layered ASP.NET Core 8 Web API using Domain, Application, Infrastructure, and API projects.
Implemented product, category, basket, order, account, and payment modules with clean controller/service/repository separation.
Designed SQL Server persistence with Entity Framework Core, migrations, entity configurations, and seed data.
Added authentication and authorization using ASP.NET Core Identity and JWT bearer tokens.
Built Redis-based shopping basket storage with expiration support for cart sessions.
Integrated Stripe Payment Intents to create and update payments based on basket totals and delivery costs.
Applied Repository, Unit of Work, and Specification patterns for reusable data access, filtering, sorting, pagination, and eager loading.
Used AutoMapper and DTOs to separate API contracts from domain entities.
Added Swagger/OpenAPI documentation with JWT authorization support.
Implemented global exception handling middleware for consistent API error responses.
